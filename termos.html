Termos de uso

Termos e Condições.

1. Política de Estorno e pagamento

1.1 Não aceitamos estornos dos valores após o pagamento ser realizado. Pois todo projeto é analisado e acordado antes do cliente efetuar o pagamento.

1.2 Não aceitamos pagamentos via cartão de crédito, isso inclui: sites e manuntenção mensal.

1.3 Não aceitamos pagamentos via boleto bancário. Todos os pagamentos serão feitos através de PIX.

2. Informações do usuário

2.1 Todo cliente está ciente que precisamos dos dados para cosneguir realizar as configurações necessárias para o desenvolvimento do site. Se caso for necessário, o cliente concorda em passar dados suficientes para que possamos dar continuidade no processo.

3. Planos

3.1 Os planso mensais serão seguidos conforme descrição. Toda alteração que quiser ser feita além do plano, será cobrado uma taxa maior ou igual ao valor da taxa mensal fixada no site.
